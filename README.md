# いしのこえWS

いしのこえWSは、MATHRAXの作品「いしのこえ」を紹介・解説しながら、石に触れると音を奏でるちいさな「いしのこえ」をつくるワークショップです。
自分のピンとくる1つの石を選んだら、その相棒となる2つ目の石を選びます。
ここにあるプログラムは、タッチすると音を奏でるプログラムです。

Seeduino XIAOにプログラムします。
